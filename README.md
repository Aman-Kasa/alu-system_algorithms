# 🧠 ALU System Algorithms

> Advanced data structures and algorithms implemented in C — covering trees, graphs, pathfinding, and compression.

---

## 📂 Projects

| Folder | Topic | Key Algorithms |
|---|---|---|
| [`0x00-red_black_tree`](./0x00-red_black_tree/) | Red-Black Tree | Insert, Delete, Validate, Rotate |
| [`0x04-nary_trees`](./0x04-nary_trees/) | N-ary Tree | Insert, Traverse, Diameter, Path |
| [`graphs`](./graphs/) | Graph (Adjacency List) | DFS, BFS, Add Vertex/Edge |
| [`huffman_coding`](./huffman_coding/) | Huffman Compression | Priority Queue, Huffman Tree & Codes |
| [`pathfinding`](./pathfinding/) | Pathfinding | Backtracking, Dijkstra, A\* |

## ⚙️ Language & Standards

- **C** (C99 / GNU99)
- Compiled with `gcc` using `-Wall -Wextra -Werror -pedantic`
- No external libraries — pure standard C
